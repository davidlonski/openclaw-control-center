🚀 Setup Checklist

1
Install OpenClaw
Install via npm: npm install -g openclaw
Then run: openclaw onboard to configure your first agent and gateway.
2
Start the Gateway
Run: openclaw gateway start
Or set it up to auto-start: openclaw gateway install-service
3
Configure at least one agent
Edit ~/.openclaw/openclaw.json and add an agent to agents.list.
Or use: openclaw agents add
4
Discord (optional)
Add a Discord bot token to channels.discord.accounts.default.token in your config.
Create a bot at discord.com/developers.
5
Run the install script (macOS)
Sets up LaunchAgents for auto-start on boot:
bash /path/to/Fredrick-CLAW/install.sh
→ Open Dashboard
